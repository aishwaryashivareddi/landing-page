# Landing Page

This project is a fully responsive landing page created as part of a Web Development assignment.  
The design was replicated from a given Figma file using **HTML**, **CSS**, and **Vanilla JavaScript** — no frameworks used.

##  Features
- Fully responsive layout  
- Mobile navigation menu  
- Smooth scroll  
- Clean and semantic HTML  
- Organized folder structure (HTML/CSS/JS/Images)  

##  Folder Structure
Landing_page/
│── index.html  
│── css/  
│     └── style.css  
│── js/  
│     └── script.js  
└── images/  
      └── assets  

##  How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser
